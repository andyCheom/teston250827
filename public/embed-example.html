🤖 GraphRAG 챗봇 위젯 임베드 예제

이 페이지는 GraphRAG 챗봇 위젯을 다른 웹사이트에 임베드하는 방법을 보여줍니다.

💡 기본 사용법

가장 간단한 방법으로 위젯을 추가하려면 다음 코드를 HTML의 </body> 태그 직전에 추가하세요:

<!-- GraphRAG 챗봇 위젯 --> <script src="https://your-domain.com/embed-widget.js"></script>

⚙️ 사용자 정의 설정

위젯의 동작을 사용자 정의하려면 설정 객체를 제공할 수 있습니다:

<script> // 위젯 설정 (스크립트 로드 전에 정의) window.GraphRAGWidgetConfig = { apiBaseUrl: 'https://your-api-domain.com', firebaseProjectId: 'your-project-id', version: '1.0.0' }; </script> <script src="https://your-domain.com/embed-widget.js"></script>

🔧 프로그래밍 방식 초기화

더 많은 제어가 필요한 경우 프로그래밍 방식으로 위젯을 초기화할 수 있습니다:

<script src="https://your-domain.com/embed-widget.js"></script> <script> // DOM이 로드된 후 수동 초기화 document.addEventListener('DOMContentLoaded', function() { window.initGraphRAGWidget({ apiBaseUrl: 'https://your-api-domain.com', firebaseProjectId: 'your-project-id' }); }); </script>

✨ 위젯 기능

  • ✅ 완전히 독립적인 스타일링 (기존 사이트와 충돌 없음)
  • ✅ 반응형 디자인 (모바일 친화적)
  • ✅ Firebase Firestore 대화 저장
  • ✅ GraphRAG API 완전 지원
  • ✅ 실시간 타이핑 인디케이터
  • ✅ 대화 세션 관리
  • ✅ 크로스 브라우저 호환성

⚠️ 중요 사항

  • API 엔드포인트가 CORS를 허용하도록 설정되어야 합니다
  • Firebase 프로젝트가 공개 액세스를 허용해야 합니다
  • HTTPS에서만 정상적으로 작동합니다
